A modelling-focused project that predicts outcomes based on structured event data and demonstrates end-to-end analytical thinking

this is to display pipeline thinking. It focuses on modelling logic instead of writing models

steps for this project
1. get a simulated data from simulate_data.R
2. make the simulated data more fit to reality using feature_engineering.R such that 
   features now have business purposes, i.e. short -> quick event, medium -> moderate, long -> prolonged
3. using logistic regression for binary responses 0 and 1
